# 🧠 Project Overview

This is a full-stack portfolio web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).

The application demonstrates:
- Responsive frontend design
- RESTful API integration
- Dynamic project showcase
- Contact form integration
- Secure backend architecture

Frontend and backend are deployed separately to simulate real-world production deployment architecture.
